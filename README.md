## 🚀 Quick Start

### Prerequisites
- Python 3.8+

### Run Locally

1. **Install Dependencies**
   ```bash
   pip3 install -r requirements.txt
   ```

2. **Start the App**
   ```bash
   ./run.sh
   # OR
   python3 -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
   ```

3. **Open Browser**
   - Visit [http://localhost:8000](http://localhost:8000)
