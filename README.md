# Alchemy AI - Turn Prospects Into CUSTOMERS

Alchemy AI is a suite of 5 specialized AI agents working 24/7 to grow your business. Our agents handle lead generation, outreach, and content repurposing with zero management required.

## Key Features

- **Lead Finder Agent**: Discover qualified prospects across 150+ sources automatically.
- **Lead Outreach Agent**: Send personalized multi-channel messages at scale.
- **Content Repurposing Agent**: Transform one piece of content into 10+ formats instantly.
- **CMS Dashboard Agent**: Unified control center for all agents and campaigns.
- **SEO Copilot Agent**: Automated audits and optimizations for higher search rankings.

## Technology Stack

- **HTML5/CSS3**: Modern, responsive layout with glassmorphism and advanced animations.
- **JavaScript**: Smooth interactions and dynamic UI effects.
- **Intersection Observer API**: Scroll-triggered animations.

## Getting Started

Since this is a static website, you can view it directly by opening `index.html` in your browser.

1. Clone the repository:
   ```bash
   git clone https://github.com/Daaksh05/alchemy-reachout-engine.git
   ```
2. Open `index.html` in any modern web browser.
